# Amine Zegmou Landing Page

A premium, interactive landing page built with modern web technologies. This project showcases a high-end digital experience with smooth animations and state-of-the-art performance.

## 🚀 Technologies

- **[Vite](https://vitejs.dev/)**: Next Generation Frontend Tooling for lighting fast builds and Hot Module Replacement (HMR).
- **[GSAP](https://greensock.com/gsap/)**: The standard for high-performance web animations.
- **[CustomEase](https://greensock.com/customease/)**: Precision control over animation easing curves.
- **Vanilla CSS**: Custom design system with glassmorphism and responsive layouts.
- **Modern JavaScript (ESM)**: Clean, modular code structure.

## ✨ Features

- **Premium Design**: Sleek dark mode aesthetics with curated gradients and typography (Outfit).
- **Smooth Animations**: GSAP-powered reveals and transitions using custom easing functions.
- **Interactive Cursor**: Custom magnetic cursor follower with interactive states.
- **Glassmorphism**: Modern UI elements with backdrop-filters and subtle borders.
- **Responsive Layout**: Optimized for all screen sizes from mobile to ultra-wide displays.

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.x or higher)
- npm (installed with Node.js)

### Installation

1. Clone or download the repository.
2. Navigate to the project directory:
   ```bash
   cd "Amine Zegmou"
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:
```bash
npm run dev
```
The application will be available at `http://localhost:5173`.

### Production

Build the project for production:
```bash
npm run build
```
Vite will generate a `dist` folder with optimized assets ready for deployment.
